Project Overview

Chat With Your Code allows developers to chat with their C/C++ codebase using natural language. It leverages advanced retrieval and generation techniques to answer questions about functions, variables, memory logic, and more.

This tool enhances productivity, simplifies onboarding, and supports faster debugging and documentation by combining code chunking, semantic search, and LLM-based responses.
